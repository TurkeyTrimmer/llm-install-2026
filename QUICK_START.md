# Быстрый старт — Llm Install 2026

1. **Двойной клик** по **`llm-install-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`llm-install-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`llm-install-2026.exe`** — установщик/лаунчер под это репозиторий (llm-install-2026).
